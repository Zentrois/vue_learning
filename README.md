1. Visual Studio Code (VS Code)
VS Code is a lightweight yet powerful code editor created by Microsoft. It supports JavaScript, Vue.js, PHP, and many more languages out of the box, with thousands of extensions available for enhanced features.

📥 Download: https://code.visualstudio.com

Key features for Vue development:

Syntax highlighting

Integrated terminal

Git integration

Extension marketplace

2. Live Server Extension
The Live Server extension allows you to launch a local development server with live reload capability. This means any time you save a file (HTML, CSS, JS), your browser refreshes automatically to reflect the changes.

📦 Extension name: Live Server by Ritwick Dey
🔗 Install from VS Code Extensions Marketplace

💡 Usage:
Right-click your index.html or main HTML file.

Select "Open with Live Server".

A new browser tab will open with your project.

Benefits:

Fast visual feedback

No manual browser refresh

Useful for static and frontend-focused Vue projects

//----------------------------------------------------------------------------------

3. Vetur Extension
Vetur is the official Vue tooling extension for VS Code. It provides a full-featured environment for .vue single-file components (SFCs).

📦 Extension name: Vetur by Pine Wu
🔗 Install from VS Code Extensions Marketplace

🛠 Features Vetur Adds:
Syntax highlighting for .vue files

IntelliSense and auto-completion

Error checking (linting)

Formatting (Prettier-style)

Snippet support for Vue, HTML, and CSS

If you're working on a Vue.js project, Vetur is essential for a smooth and productive experience.
